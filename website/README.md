# TokioAI Website

Sitio estatico (HTML/CSS/JS) para describir el proyecto.

## Abrir local

Opcion simple:
- abrir `website/index.html` en el navegador

Opcion recomendada (servidor local):

```bash
cd website
python3 -m http.server 5173
```

Luego:
- http://127.0.0.1:5173

## Antes de publicar

- Link de GitHub: `https://github.com/TokioAI`
- Ajustar textos de la fundacion si necesitas datos legales adicionales.
- El sitio usa el logo en `website/tokio-spiral.png` (reemplazalo por tu asset oficial si corresponde).
